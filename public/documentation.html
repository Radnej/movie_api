MyMovies Application

This is the server-side component of a “movies” web application.
The web application will provide users with access to information about different movies,
directors, and genres. Users will be able to sign up, update their personal information,
and create a list of their favorite movies.

Business Logic URL HTTP Method Query parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None None Array of JSON objects holding data about all movies
Return data about a single movie by title to the user /movies/:Title GET :Title None A JSON object holding data about the selected movie containing description, genre, director and img URL. Example: 
          
            {
              Title: '', 
              Description: '',
              Year: '', 
              Genre: {
                Name: '',
                Description: ''
              } 
              Director: {
                Name: '', 
                Birth: '',
                Death: '',
                Bio: ''
              },
              Actors: {
          
              }
              ImagePath: ''
            }
Return data about a genre (description) by name (e.g., “Thriller”) genres/:Name GET :Genre None A JSON object holding data about the selected genre. Example: 
          
            {
              "Name": " ",
              "Description": " "
          }
Return data about a director (bio, birth year) by name /directors/:Name GET :Name None A JSON object holding data about the selected director. Example: 
          
            {
              "Name": "",
              "Bio": "",
              "Birth": "Date",
              "Death": "Date"
          }
Allow users to update their user info (username) /users/:Username PUT :Username A JSON object holding the new username. Example: 
          
            {
              "Username": " ",
              "Password": " ",
              "Email": " ",
              "Birthday": "Date",
          }
A JSON object with the updated user profile. Example: 
          
            {

              "Password": " ",
              "Email": " ",
              "Birthday": "Date",

Allow users to add a movie to their list of FavoriteMovies /users/:Username/movies/:MovieID POST :Username, :MovieID None It will return the updated user object containing the added favorite movie
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE :Username, :MovieID None It will return the updated user object containing the added favorite movie
